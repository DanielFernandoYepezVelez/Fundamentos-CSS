OVERFLOW:
Es una propiedad que controla como se va a comportar la caja con el contenido
que se desborde de ella.

Es un shorthand que engloba =>
overflow-x
overflow-y

Tiene tres valores posibles =>
hidden: Oculta todo el contenido que se desborde.
auto: Muestra la barra de scroll solo si hace falta.
scroll: Muestra ambas barras de scroll independientemente de si necesitan.

Los overflow tienen diferentes funciones
y se explican a continuación:

overflow: hidden; => Sirve para ocultar algo que se desborde
de un elemento con un alto y ancho definido.

overflow: visible; => Viene por defecto como valor de 
la propiedad de overflow.

overflow: scroll; => sirve para darle scroll arriba y abajo
además de la derecha e izquierda a un elemento.

overflow-y: scroll; => sirve para darle scroll solamente 
arriba y abajo un elemento.

overflow: auto; => sirve para definir un scroll de forma vertical.
